# Changelog

## [1.0.0] — 2026-06-29

### Добавлено
- Боковая панель Chrome с полноценным чатом Pi
- Стриминг ответов ассистента в реальном времени
- Динамический индикатор активности: "Агент думает…" → "Выполняет: {tool}…" → "Пишет ответ…"
- DOM Picker — интерактивный выбор элементов страницы для отправки в Pi
- Markdown-рендеринг ответов ассистента
- Отображение информации о модели и использовании контекста
- Команда `/chrome-assistent-connect` для Pi

### Архитектура
- Event-driven синхронизация (snapshot при connect, все обновления через events)
- Direct WebSocket session server на localhost (без промежуточного broker)
- Паттерн mirror-server по образцу pi-web-ui
