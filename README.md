# Pi Chrome Extension

Расширение для браузеров на ядре Chrome, которое позволяет подключить [Pi Coding Agent](https://github.com/earendil-works/pi) в качестве ассистента прямо из браузера.

## Зачем это нужно

Вы сможете отправлять сообщения Pi Coding Agent напрямую из вашего открытого терминала, в котором запущен `pi`. Это даёт возможность удобно взаимодействовать с агентом, не переключаясь между окнами.

## Как это работает

1. Расширение устанавливается в браузер и предоставляет интерфейс для общения с Pi.
2. Сообщения из расширения передаются в запущенный экземпляр `pi` в терминале.
3. Ответы агента отображаются обратно в интерфейсе расширения.

## Требования

- Браузер на ядре Chrome (Chrome, Edge, Brave и др.)
- [Pi Coding Agent](https://github.com/earendil-works/pi), установленный и запущенный в терминале
