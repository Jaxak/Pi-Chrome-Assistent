# Pi Chrome Assistent

## Что это

Pi Chrome Assistent — локальная связка Chrome-расширения и Pi Coding Agent. Она позволяет выбрать Pi-сессию, запустить DOM picker на текущей вкладке и отправить в Pi URL страницы, заголовок, текст, HTML-фрагмент, CSS-селектор и комментарий.

Текущий пользовательский сценарий построен вокруг трёх вкладок popup:

- **Асистент** — основная кнопка **«Отправить в Pi»**;
- **Сессии** — выбор активной Pi-сессии и просмотр диагностики;
- **Авторизация** — получение токена браузера и привязка браузера к Pi.

Важно: ответы Pi не возвращаются в popup. После отправки продолжайте читать ответ в терминале, где запущен `pi`.

## Быстрый старт

1. Установите зависимости и соберите расширение:
   ```bash
   npm install
   npm run build:chrome
   ```
2. Откройте `chrome://extensions`, включите режим разработчика и загрузите папку `dist/chrome` как неупакованное расширение.
3. Запустите `pi` в нужном проекте.
4. Если локальное Pi-расширение было добавлено после старта `pi`, выполните:
   ```text
   /reload
   ```
5. Откройте popup расширения и перейдите на вкладку **«Авторизация»**.
6. Нажмите **«Скопировать токен»** или скопируйте показанный токен вручную.
7. В Pi выполните:
   ```text
   /chrome-assistent-auth
   ```
   и вставьте токен из вкладки **«Авторизация»**.
8. В нужной Pi-сессии выполните:
   ```text
   /chrome-assistent-connect [alias]
   ```
9. В popup откройте вкладку **«Сессии»**, выберите нужную цель Pi.
10. Вернитесь на вкладку **«Асистент»** и нажмите **«Отправить в Pi»**.

Под капотом браузер работает с локальным брокером `ws://127.0.0.1:17345`, а Pi хранит runtime-state в `~/.pi/chrome-assistent/`.

## Ограничения текущей версии

- Транспорт рассчитан только на локальный broker `127.0.0.1:17345`.
- Ответы Pi остаются в терминале и не стримятся обратно в браузер.
- Для начала работы нужны два шага: сначала `/chrome-assistent-auth`, затем `/chrome-assistent-connect`.
- Если сессия Pi, владеющая broker, завершилась, живую сессию может потребоваться заново подключить командой `/chrome-assistent-connect`.
